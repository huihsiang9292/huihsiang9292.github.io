Blog 環境配置

部署hugo網站 (一)

hugo + github pages + github action + vscode + wsl2

2023/05/15

配置環境

Ubuntu

sudo apt update && sudo apt upgrade -y

Golang

sudo apt install golang-go -y
go version

Git

sudo apt install git -y
git version

Hugo

sudo snap install hugo
hugo version

建立 Github 儲存庫

私有儲存庫

用來存放所有原始檔案，只有你看得到
Repository name 可以隨意取

公開儲存庫

用來存放public裡的檔案，所有人都看得到
Repository name 要填上YourName.github.io

配置hugo-theme-stack主題

打開 vscode 後，clone 私有儲存庫

因為資料夾已經存在，加上 --force 強制執行

hugo new site YourBlog --force
cd www
hugo mod init github.com/YourName/YourName.github.io

在 config.toml 中寫入:

[[module.imports]]
path = "github.com/CaiJimmy/hugo-theme-stack/v3"

在本地預覽通常都會加上 -D ，這樣才能顯示草稿

hugo server -D


目前什麼都沒有，所以我們要加入主題提供的範本

  • content
    把 exampleSite 下的 content 資料夾複製到目錄下
  • config
    把 config 資料夾複製到目錄下，刪除 config.toml


從 windows 複製到 wsl2 可能會出現帶有 .Identifier 的檔案，全部刪掉就行了

配置 Github Action

說明

每次寫完文章都要手動發佈有點麻煩，配置好 action 後，只需要在寫完文章後 push 到 github，網站就會更新

  • actions/checkout : 官方 action，讓 workflow 可以存取 code
  • peaceiris/actions-hugo : 配置 Hugo 環境
  • peaceiris/actions-gh-pages : 將某個資料夾下的檔案 push 到特定 branch，也支援 push 到別的 repository

建立 action

建立 action 資料夾，加上 -p 讓它可以一次建立兩層

mkdir -p .github/workflows/

name: Deploy Hugo site to Pages

on:
  push:
    branches:
      - main

  workflow_dispatch:

jobs:
  build:
    runs-on: ubuntu-latest

    steps:
    - uses: actions/checkout@v3

    - uses: peaceiris/actions-hugo@v2
      with:
        hugo-version: 'latest'
        extended: true

    - run: hugo --gc --minify

    - uses: peaceiris/actions-gh-pages@v3
      with:
        deploy_key: ${{ secrets.ACTIONS_DEPLOY_KEY }}
        external_repository: YourName/YourName.github.io
        publish_branch: main
        publish_dir: ./public

Deploy Key

生成密鑰

ssh-keygen -t rsa -b 4096 -C "$(git config YourGithubMail)" -f key -N ""


key 是私鑰，key.pub 是公鑰

私鑰

cat key

將顯示出來的內容全部複製，包含 -----BEGIN OPENSSH PRIVATE KEY----- -----END OPENSSH PRIVATE KEY-----，放在私有儲存庫
名稱 : ACTIONS_DEPLOY_KEY

公鑰

cat key.pub

將顯示出來的內容全部複製，放在公開儲存庫
名稱 : Public key of ACTIONS_DEPLOY_KEY

部署到 Github Pages

git config --global user.name YourName
git config --global user.email YourMail@example.com

push 上 github，打開瀏覽器輸入 YourName.github.io 就可以看到網站啦

SFML 環境配置

2023/06/04

下載 MinGW、SFML

MinGW Builds 7.3.0 (64-bit)、GCC 7.3.0 MinGW (SEH) - 64-bit
如果要使用自己的 MinGW ，版本必須與 SFML 相同

配置環境變數

  1. 解壓縮後，將 x86_64-7.3.0-release-posix-seh-rt_v5-rev0 重新命名成 mingw64-7.3.0
  2. 複製 mingw64-7.3.0、SFML-2.5.1 中 bin 的路徑
  3. 設定 > 系統 > 系統資訊 > 進階系統設定 > 進階 > 環境變數 > Path
  4. 將複製的路徑新增至 Path 中

例如 : C:\sfml\mingw64-7.3.0\bin、C:\sfml\SFML-2.5.1\bin

    #include <SFML/Graphics.hpp>

    int main()
    {
        sf::RenderWindow window(sf::VideoMode(200, 200), "SFML works!");
        sf::CircleShape shape(100.f);
        shape.setFillColor(sf::Color::Green);

        while (window.isOpen())
        {
            sf::Event event;
            while (window.pollEvent(event))
            {
                if (event.type == sf::Event::Closed)
                    window.close();
            }

            window.clear();
            window.draw(shape);
            window.display();
        }

        return 0;
    }

配置 VScode

  1. 將滑鼠移至 #include <SFML/Graphics.hpp> 的紅色波浪上，點擊 快速修復、編輯"includePath"設定
  2. 將 編譯器路徑 改為 C:/sfml/mingw64-7.3.0/bin/g++.exe
  3. 在 包含路徑 新增一行 C:\sfml\SFML-2.5.1\include
  4. 編譯之後會報錯，在自動建立的 .vscode/tasks.json 中修改

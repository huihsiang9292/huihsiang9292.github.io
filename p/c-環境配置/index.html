更新

pacman -Syu

下載 mingw-w64

pacman -S mingw-w64-ucrt-x86_64-gcc

下載 mingw-w64 工具組

pacman -S --needed base-devel mingw-w64-x86_64-toolchain

新增環境變數

  1. 複製 mingw-w64 中 bin 的路徑 ( 預設安裝路徑 : C:\msys64\mingw64\bin )
  2. 設定 > 系統 > 系統資訊 > 進階系統設定 > 進階 > 環境變數 > Path
  3. 將 C:\msys64\mingw64\bin 新增至 Path 中
